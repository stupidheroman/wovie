# Installation
**Server Requirements**
**Checking to ensure that your web host have the minimum requirements to run Wovie Script**
- PHP >= 5.6.0
- allow_url_fopen
- cURL
- PDO
- GD Extension
- mbstring Extension
- EXIF Extension
